# 说明
检测道路事故。

# 检测功能
## 1. 群体性检测
抛洒物检测，事故检测
## 2. 单体性检测
异常急刹，应急车道占用，异常停车，超速行驶，低速行驶
## 3. 直接数值检测
拥堵检测
